# Gallery photos

Drop your own photos here, named:

```
gallery-1.jpg
gallery-2.jpg
...
gallery-8.jpg
```

They appear automatically in the Gallery section. Until a file exists,
an elegant "Photo N" placeholder tile is shown in its place, so the
layout always looks intentional.

Tips:
- Square-ish images look best (the grid uses a 1:1 aspect ratio).
- ~1000–1600px wide is plenty; keep each file under ~500 KB if you can.
- `.jpg` is expected. To use `.png`/`.webp`, update the extension in the
  gallery `src` in `src/App.tsx`.
